About
I was having lunch with a couple of developers when I visited New York. One of the developers mentioned making a "To Do" list and at the time I didn't think much of it. However, I kept thinking about it and I knew I had to make one.
Code
I love doing a tutorial and taking what I learned and making it better, and coding more efficiently. I knew that I wanted a way to display it, check it off, or delete it.
Design
Red. Simple. Attention. Urgent. All red colors were used to grab the users attention. I use "aquamarine" when coding to make sure everything is working. When I used it with the check mark I actually enjoyed it! It wasn't a simple green which I liked so I decided to keep it.
Next
I would like to build this app using React next. I have considered making this a fullstack project and adding Node in as well.
