About
I was having lunch with a couple of developers when I visited New York. One of the developers mentioned making a "To Do" list was pretty much standard in learning JavaScript. Almost like a rite of passage. So, I decided to go out and make one.
Code
This project really helped with using append in JavaScript. I usually hard code what I want to see in in the app and go from there. Along with displaying the user input, I wanted a way to check it off, and delete it.
Design
Red. Simple. Attention. Urgent. All red colors were used to grab the users attention. I use "aquamarine" when coding to make sure everything is working. When I used it with the check mark I actually enjoyed it! It wasn't a simple green which I liked so I decided to keep it.
